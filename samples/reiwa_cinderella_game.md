# 令和版シンデレラ - ゲームモード設定

## 基本設定

### 舞台
2025年の東京（渋谷・六本木・表参道）

### 主人公: シンデレラ（23歳）
- **職業**: ITベンチャーでインターン（雑用係）
- **夢**: デザイナーになりたい
- **性格**: 優しくて前向き、でも疲れてる
- **住まい**: シェアハウス（ルームメイトに虐げられている）
- **SNS**: インスタのフォロワー300人（地味）

### イケメン社長（28歳）
- **職業**: 急成長中のスタートアップCEO
- **性格**: 見た目はチャラいけど中身は真面目
- **趣味**: クラブイベント、アート鑑賞
- **目的**: 本物の愛を探している（周りは金目当ての女性ばかり）

### 意地悪ルームメイト2人
- **リーダー格**: インフルエンサー気取り（フォロワー5万）
- **取り巻き**: リーダーにベッタリ
- **行動**: シンデレラに家事を押し付け、マウント取りまくり

### 妖精役: 謎のスタイリスト（30代）
- **正体**: 元トップモデル、今は隠れて活動
- **能力**: メイク、ファッション、SNS戦略の天才
- **モットー**: 「本当の美しさは内面から」

## ゲームストーリー展開

### ステージ1: いじめられる日々
**シーン: シェアハウス（夜）**

シンデレラは毎日残業でクタクタ。帰宅すると意地悪ルームメイトから「洗濯まだ？」「掃除サボってない？」と詰められる。

ある日、会社のSlackに通知が来る：
```
【社内イベント】六本木VIPパーティー🎉
日時: 明日夜8時
場所: ROPPONGI CLUB LUXE
ドレスコード: フォーマル・エレガント
※社長が参加予定
```

ルームメイトたちは大騒ぎ。シンデレラは「私も行きたいな…」とつぶやくが、「お前が行ってどうすんの？ww」と笑われる。

**プレイヤー選択肢:**
- A: 諦める（BAD END直行）
- B: こっそり準備する（ゲーム続行）
- C: 直接対決する（難易度UP）

---

### ステージ2: 謎のスタイリストとの出会い
**シーン: 表参道カフェ（昼休み）**

落ち込んだシンデレラがカフェで涙ぐんでいると、隣の席から声をかけられる。

謎の女性:「ねえ、あなた…すごいポテンシャル持ってるわよ。でも埋もれてる」

女性は元トップモデルのスタイリスト。シンデレラの内面の美しさを見抜き、パーティー用の変身を提案。

**提供してくれるもの:**
```
✅ ヘアメイク（プロ級）
✅ ドレス（デザイナーズブランド）
✅ アクセサリー（一式）
✅ Uber予約（行きと帰り）
✅ Instagram映えテクニック
```

**条件:**
```
⚠️ 23:30までに会場を出ること
理由: 魔法じゃなくて、Uberの予約時間（23:45発）
遅れたら全部レンタル料金が高額請求される
```

**プレイヤー選択肢:**
- A: 受け入れる（ゲーム続行）
- B: 怪しむ（追加イベント発生）
- C: 断る（GAME OVER）

---

### ステージ3: 六本木VIPパーティー
**シーン: ROPPONGI CLUB LUXE（夜8時半）**

会場は超豪華。IT業界の有名人がズラリ。シンデレラは変身した姿で登場し、周囲の視線を独占。

イケメン社長が近づいてくる:「初めまして。お名前は？」

2人は意気投合。DJ BLASTのクラブミュージックに合わせて踊り、ルーフトップバーで東京の夜景を見ながら語り合う。

**ミニゲーム要素:**
```
1. 会話選択肢（社長の好感度UP/DOWN）
2. ダンスリズムゲーム（盛り上がり度UP）
3. インスタストーリー投稿タイミング（バズるかどうか）
```

**時間経過:**
```
20:30 - 入場
21:00 - 社長と出会い
22:00 - ルーフトップで会話
23:00 - 最高潮の盛り上がり
23:15 - ⚠️ 時間制限警告
23:30 - タイムリミット！
```

**プレイヤー選択肢（23:15）:**
- A: 23:30前に退出（正解ルート）
- B: もう少し残る（リスクルート）
- C: 連絡先を交換してから退出（ベストルート）

---

### ステージ4: 忘れ物事件
**シーン: パーティー会場 → 渋谷の街（深夜）**

シンデレラは慌てて会場を飛び出し、Uberに飛び乗る。しかし、会場の階段に **限定スニーカー（片方）** を落としてしまう。

翌日、社長はスニーカーの持ち主を探すため、社内で「この靴の持ち主を探しています」とSlackで告知。

ルームメイトたちは「私のだ！」と主張するが、サイズが合わない。

**プレイヤー選択肢:**
- A: 黙っている（BAD END）
- B: 「私のです」と名乗り出る（TRUE END）
- C: こっそり社長にDMを送る（隠しルート）

---

### ステージ5: 真実の告白
**シーン: 会社の屋上（昼休み）**

シンデレラが名乗り出ると、社長は驚く。「君が…あのインターンの？」

シンデレラ:「はい。あの夜は…本当の私じゃなかったかもしれません。でも、あなたと話した時間は本物でした」

社長:「外見なんてどうでもいい。君の話し方、考え方、優しさに惹かれたんだ。一緒に働かないか？デザイナーとして」

**エンディング分岐:**

#### TRUE END: 夢のデザイナーデビュー
- シンデレラはデザイナーとして採用される
- 社長とは仕事仲間として良い関係を築く
- ルームメイトとは距離を置き、新しいマンションへ引っ越し
- 1年後、自分のブランドを立ち上げる

#### LOVE END: 社長と恋人に
- デザイナー採用 + 社長と交際開始
- 半年後、社長の会社のブランディングを担当
- ルームメイトは嫉妬するも、シンデレラは許す
- 2人で新しいスタートアップを共同創業

#### HIDDEN END: インフルエンサーとして成功
- パーティーのインスタ投稿がバズる（100万いいね）
- デザイナー兼インフルエンサーとして活動
- 社長とは良き理解者として友情関係
- 「本当の自分で生きる」がテーマのブランドを展開

## ゲームモード要素

### パラメータ管理
```
- 好感度（社長）: 0-100
- 自己肯定感: 0-100
- SNSフォロワー数: 300 → ???
- 疲労度: 0-100（高いとBAD END）
- ファッションセンス: 0-100
```

### ミニゲーム
```
1. 会話選択（選択肢によって好感度変動）
2. ダンスリズムゲーム（盛り上がり度）
3. コーディネート選び（ファッションセンスUP）
4. SNS投稿タイミング（バズるかどうか）
5. 時間管理（23:30までに退出）
```

### 収集要素
```
- ファッションアイテム（レンタル or 購入）
- 名刺コレクション（業界人との出会い）
- インスタストーリー（思い出アルバム）
- 励ましのメッセージ（スタイリストから）
```

## システムプロンプト（ゲームモード用）

```
あなたは「令和版シンデレラ」のゲームマスターです。
プレイヤーはシンデレラとなり、選択肢を選んで物語を進めます。

ルール:
1. 各シーンで選択肢を2-4個提示
2. 選択によってパラメータが変動
3. ギャル語で状況を説明（2-4文）
4. 重要な場面では「⚠️警告」を出す
5. エンディングは複数用意（TRUE/LOVE/HIDDEN/BAD）

口調:
- 一人称: あたし（ゲームマスター視点）
- 二人称: キミ、プレイヤー
- 絵文字: 各返答の最後に1つ

状況説明と選択肢を必ずセットで提示してください。
```
